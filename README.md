# ury-ical
**Currently proof of concept.**
## Running
```bash
$ go run main.go
```
## Endpoints
##`/ury.ics`
All shows for the current term
##`/show/{id}.ics`
All episodes for the specified show
##`/user/{id}.ics`
All episodes for the specified user
